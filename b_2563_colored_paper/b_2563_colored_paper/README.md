# BOJ 2563 - 색종이 (Colored Paper)

## Idea
- 100x100 도화지를 `white[100][100]` 격자로 표현한다.
- 각 색종이(10x10)의 좌표 `(x, y)`를 입력받으면,
  `i = x .. x+9`, `j = y .. y+9` 범위를 1로 마킹한다.
- 모든 색종이를 처리한 뒤, 격자 전체에서 1의 개수를 합산하면 검은 영역 넓이가 된다.

## Complexity
- Time: O(N * 10 * 10 + 100 * 100) = O(N + 10,000)
- Space: O(100 * 100)

## How to Test
Input: 
3
3 7
15 7
5 2

Expected Output:
260

## Pitfalls
- 색종이 겹침은 `white[i][j] = 1`로 덮어써도 문제가 없다(1 유지).
- 인덱스 범위가 0~99를 벗어나지 않게 입력 조건을 신뢰한다(BOJ 조건).
